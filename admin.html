⚠️ Firebase এখনো সেটআপ হয়নি নিচের ধাপ অনুসরণ করুন:
  1. console.firebase.google.com এ যান, Google দিয়ে লগইন করুন
  2. "Add project" → প্রজেক্টের নাম দিন → তৈরি করুন
  3. Project Settings (⚙️ আইকন) → "Your apps" → Web (</>) ক্লিক
  4. Config কোডের apiKey, projectId ইত্যাদি কপি করুন
  5. এই admin.html ফাইলে FIREBASE_CONFIG অংশে বসান
  6. Firestore Database → "Create database" → Test mode → Enable
